What is Quiz Game?

This is a fun game our group make where you can enjoy answering some seriously tricky questions

How to play the game?

To start playing, press new game in the main menu. There are eight 4-choice questions.

If you want to restart the game, press Reset.

You can press the Hint button to remove 2 incorrect choices but be careful, You can use 2 hints only!

How the game work?

By pressing the new game button, you have to answer all questions correctly.

If you answer one of the question incorrectly, you lose :D
